Let's talk about CloudWatch.

  • Hosted metrics.
  • All of the AWS services have support built in.
  • A key part of Autoscaling.

Metrics

  • Namespaces.
  • More or less just timestamped counts.
  • But you can specify the units.
  • 1-minute time intervals.
  • Need data about less than a minute? Use statistic sets

Custom Metrics Benefits

  • Sometimes you need to measure something domain specific.
  • You don't have to maintain your own statsd-like service.
  • Useful graphing tool in the AWS console.
